# JogoVelha-Jquery

Jogo da Velha realizado por Donato C. Ávila, durante treinamento na empresa SMN.
Realizado em HTML5, CSS3 e jQuery. 

Funcionalidade:

	- É possível escolher o jogador.
	- Quem começa a primeira partida.
	- O seu icone (Xis ou Bola).
	- Marca pontuação.
	- Volta ao inicio.
	- Zera placar.
